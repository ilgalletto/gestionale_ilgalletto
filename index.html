Calendario

Il Calendario mostra tutte le prenotazioni su griglia mensile. Il mese corrente appare sempre in cima, seguito dai successivi 11 mesi.

Le colonne rappresentano i giorni del mese

Questura / Alloggiati Web
Comunicazione obbligatoria delle presenze
⚖️ Obbligo di legge — Art. 109 TULPS: le generalità degli ospiti vanno comunicate entro 24 ore dall'arrivo (6 ore per soggiorni inferiori alle 24 ore). Conservare le ricevute per 5 anni.
Come funziona il modulo
1
Imposta una volta sola il codice struttura rilasciato dalla Questura, l'username Alloggiati Web e il numero di camere disponibili
2
Dopo il check-in degli ospiti, vai in 🚔 Questura e seleziona la data
3
Seleziona gli ospiti con il checkbox e clicca ⬇️ Genera file TXT
4
Il file si scarica automaticamente. Accedi ad Alloggiati Web → Invio File → carica il file → Elabora → Invia
5
Il sistema salva automaticamente lo storico delle comunicazioni generate
📊 Stati degli ospiti
BadgeSignificato
Da inviareCheck-in confermato, comunicazione non ancora generata
⚠️ ScadutoSono passate più di 24 ore senza comunicazione
✓ ComunicatoGià incluso in un file TXT generato
📁 Il file TXT generato
Il file segue il tracciato record ufficiale della Polizia di Stato. Contiene una riga per ogni ospite:
• Tipo 18 = capofamiglia/prenotante
• Tipo 20 = familiari/accompagnatori

Per ospiti con dati mancanti appare il pulsante ✏️ che apre il check-in per completarli.
💡 Per ottenere le credenziali Alloggiati Web: contatta la Questura competente per territorio con la documentazione della struttura (licenza, CIN, documento d'identità titolare).

Impostazioni
Gestione utenti – solo Admin

La sezione Impostazioni consente di gestire gli utenti del sistema ed è accessibile solo agli Admin.

Come aggiungere un nuovo utente
1
Creare l'account su Firebase Authentication (console.firebase.google.com)
2
Aggiungere il documento in Firestore → utenti → {uid} con campi: email, nome, ruolo, stato
3
L'utente può accedere al gestionale con le proprie credenziali
⚠️ Non eliminare mai il proprio account Admin dalla sezione Impostazioni. In caso di necessità agire direttamente dalla console Firebase.
❓
Domande frequenti
Risposte ai problemi più comuni
❓ Ho inserito date sbagliate nella prenotazione. Come correggo?
Cliccare ✏️ nella riga della prenotazione e modificare le date. Il sistema controlla automaticamente le sovrapposizioni.
❓ Il pulsante Check-in non appare.
Appare solo se la data di check-in è uguale o precedente a oggi e il check-in non è ancora confermato. Verificare la data nella prenotazione e aggiornare la pagina.
❓ Ho effettuato il check-in per errore.
Contattare l'amministratore che modificherà il campo checkinConfermato direttamente su Firestore impostandolo a false.
❓ Come inserisco i dati degli ospiti aggiuntivi (non il prenotante)?
In fase di prenotazione clicca "👥 Aggiungi dati ospiti aggiuntivi" per inserirli subito (opzionale). Al momento del check-in, il sistema mostra automaticamente una scheda per ogni ospite in base al numero dichiarato: compilarle è obbligatorio per generare il file Alloggiati Web.
❓ Come genero il file per la Questura?
Dopo il check-in vai in 🚔 Questura → seleziona la data → metti il segno di spunta sugli ospiti → clicca "⬇️ Genera file TXT". Il file si scarica automaticamente. Poi aprilo su Alloggiati Web (alloggiatiweb.poliziadistato.it) → Invio File → carica → Elabora → Invia.
❓ Il file TXT viene rifiutato da Alloggiati Web.
Verificare che tutti i dati obbligatori siano presenti: cognome, nome e documento del capofamiglia. Se l'ospite mostra il badge "⚠️ documento mancante", clicca ✏️ per completare i dati e poi rigenera il file.
❓ Il calendario non mostra alcune prenotazioni.
Verificare che le date ricadano nel periodo visualizzato. Usare i pulsanti Anno prec./Anno succ. per navigare all'anno corretto.
❓ Come aggiungere un nuovo utente?
Creare l'account in Firebase Authentication, poi aggiungere il documento in Firestore con ruolo "admin", "proprietario", "visual" o "contabile" e stato "approvato". In alternativa, l'Admin può cambiare il ruolo direttamente dalla sezione Impostazioni → Utenti.
❓ Posso usare il gestionale da smartphone?
Sì, l'interfaccia è completamente responsive e si adatta a qualsiasi dimensione di schermo.
❓ Come salvare un riepilogo come PDF?
Dalla sezione Stampa, cliccare 🖨️ Stampa. Nella finestra del browser selezionare "Salva come PDF" come stampante.

Entrate
Automatiche e manuali
Entrate automatiche da prenotazioni
Vengono generate automaticamente senza inserimento manuale:
• Se il check-out è confermato → appaiono nel mese del checkout
• Se la prenotazione è solo registrata → appaiono nel mese del check-in
• L'importo usato è Importo saldato (se presente) oppure il Prezzo prenotazione

Le entrate automatiche mostrano il badge auto e non possono essere modificate.

Domande frequenti
Contabilità
❓ Le entrate delle prenotazioni non appaiono.
Verificare che la prenotazione abbia un importo nel campo Prezzo o Importo saldato. Le entrate automatiche appaiono nel mese del checkout (se confermato) o del check-in.
❓ Ho inserito una voce nel mese sbagliato.
Aprire la voce con ✏️ e modificare la data. La voce si sposterà automaticamente nel mese corretto.
❓ Come aggiungo una categoria non prevista?
Cliccare "+ Gruppo" per aggiungere un nuovo gruppo, poi "+ Conto" per i conti figli. Il piano dei conti è completamente personalizzabile.
❓ Il riepilogo annuale mostra 0 per alcuni mesi.
È normale per i mesi in cui non sono state inserite voci. I costi fissi vanno inseriti mese per mese oppure impostati come ricorrenza "Mensile".
❓ Come esporto i dati per il commercialista?
Usare il pulsante Stampa e salvare come PDF. In alternativa il commercialista può accedere direttamente con il ruolo Contabile.
